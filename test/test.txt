Solution 3 : Utiliser Visual Studio Code avec Live Server

Si vous utilisez Visual Studio Code, l'extension Live Server simplifie le démarrage d'un serveur local :

    Installez l’extension Live Server dans VS Code.
    Ouvrez votre projet dans VS Code.
    Faites un clic droit sur votre fichier HTML dans l'explorateur, puis sélectionnez "Open with Live Server".
    Votre projet s’ouvrira automatiquement dans le navigateur via un serveur local.
